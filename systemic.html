전신질환자 평가는 “조절되는가”에서 출발합니다. 각 질환을 출혈 · 감염 · 스트레스(심혈관) · 약물(상호작용·대사)의 네 축으로 보면 일관됩니다. 불확실하면 주치의 협진.

심혈관 질환

조절되는 환자는 대개 치료 가능하나 스트레스·통증·혈관수축제를 절제합니다. 응급 발생 시 응급상황 대응 참고.

고혈압

  • 조절 고혈압: 통상 치료 가능, 스트레스·통증 관리
  • BP ≥ 180/110 → 선택적 치료 연기·의뢰
  • 혈관수축제 1–2 카트리지(흡인·서서히)
  • 기립성 저혈압 주의(천천히 일으키기)

허혈성 심질환(협심증·MI)

  • 최근 심근경색(4–6주, 보수적 6개월) → 선택적 연기
  • NTG 지참 확인, 짧은 오전 진료, 불안 완화
  • 흉통 시 응급(흉통) 프로토콜

부정맥 · 심부전 · 판막

  • 페이스메이커/ICD: 전기소작 주의(초음파 스케일러 등)
  • 항응고·항혈소판 → 항혈전제 페이지
  • IE 예방요법은 고위험군만 — 약물가이드(예방요법)
혈관수축제(epinephrine) 한도 — 심혈관 고위험 환자는 1회 ~0.04mg(1:100,000 약 2카트리지)로 제한하고 흡인·분할 주입. epi 함유 치은압배사(retraction cord)는 회피합니다.

당뇨

진료 운영

  • 아침 진료(내인성 코르티솔↑), 짧게
  • 평소 식사·약 유지 — 거르고 오지 않게 사전 안내
  • 저혈당 대비(당 비치) → 응급(저혈당)
  • 조절 지표 HbA1c(<7% 양호) 참고

감염·치유

  • 감염 취약·치유 지연 → 구강위생·혈당조절이 핵심
  • 광범위 급성 감염 시 항생제 적극, 배농
  • 스테로이드·감염은 혈당↑ 유발 — 모니터
  • 치주염 ↔ 혈당조절 양방향 영향

만성신장병 · 투석

타이밍 · 출혈

  • 혈액투석 다음 날 진료(투석일은 헤파린·피로)
  • 요독성 혈소판기능 저하 → 출혈경향, 국소지혈 철저
  • 동정맥루 팔: 혈압·정맥주사 금지

약물 · 감염

  • 신배설 약물 용량조정/회피 — NSAID 회피, 일부 항생제 감량(아미노글리코사이드·아시클로비르 주의)
  • 진통은 acetaminophen 선호
  • 감염 위험↑, B/C형 간염 선별, 신성골이영양증

간질환 · 간경변

출혈

  • 응고인자↓ + 비장비대 혈소판↓ → INR·혈소판 확인
  • 관혈적 처치 전 출혈경향 평가, 국소지혈

약물

  • Acetaminophen 저용량(≤2g/일) 또는 회피
  • 아미드 국소마취제·진정제 대사 저하 → 감량/신중
  • 감염 취약, 알코올·간염력 확인

임신 · 수유

2기(4–6개월)가 선택적 치료에 가장 적합합니다. 응급·감염·통증은 어느 시기든 치료(미치료 감염이 더 위험). 앙와위저혈압 → 좌측위(우측 둔부 거상).

방사선 — 필요 시 어느 시기든 촬영 가능(저선량). 차폐(복부·갑상선)는 최신 지침상 필수 아님(과거 권장). 선택적 영상은 출산 후로 미룰 수 있음. 수유 중 대부분의 치과 약물(국소마취·acetaminophen·amoxicillin)은 사용 가능.

내분비 — 갑상선 · 부신

갑상선

  • 미조절 항진증: 카테콜아민(epi) 주의, 감염·스트레스 시 갑상선중독발작 위험 → 연기·의뢰
  • 조절되면 통상 치료 가능
  • 심한 저하증: 진정제 민감·점액부종 주의

부신부전 · 장기 스테로이드

  • 일상(비외과) 치과처치엔 스테로이드 커버 불필요
  • 1차성 부신부전 + 외과처치/큰 스트레스 시 보충 고려(처치 전 ± 24h), 대수술은 병원 IV
  • 2차·3차성은 대개 보충 불필요
  • 부신위기(저혈압·쇠약·오심) → hydrocortisone·119

출혈성 경향 · 혈액질환

항혈전제

  • 항혈소판·항응고(와파린·DOAC)는 대개 중단 없이 국소지혈
  • 상세는 항혈전제 환자 치과치료

혈액질환

  • 혈소판 < 50,000·혈우병·심한 간질환 → 혈액내과 협진·인자보충
  • 국소지혈(봉합·산화셀룰로오스·TXA), 비외상적 처치
  • 백혈병·심한 빈혈 → 감염·출혈 평가

항암화학 · 방사선 · 면역억제

화학요법

  • 호중구감소(감염)·혈소판감소(출혈) → ANC·혈소판 확인, 수치 회복기에 처치
  • 치료 시작 전 치과 정리(감염원 제거), 점막염 관리
  • 침습 처치는 종양내과 협진

두경부 방사선

  • 구강건조·우식·골괴사(ORN) 위험
  • 조사 부위 발치 회피 — 필요 발치는 방사선 전(2–3주 여유)
  • 불소 적용·구강위생, 골흡수억제제는 MRONJ

면역억제 · 이식

  • 감염관리·예방요법 개별화, 침습 전 협진
  • Cyclosporine·CCB·phenytoin → 치은비대
  • 약물 상호작용 검토

호흡기 · 신경/정신

호흡기(천식·COPD)

  • 흡입제 지참 확인, 유발자극 회피
  • 아스피린 과민 천식 → NSAID 회피, acetaminophen
  • 중증 COPD: 진정·아산화질소 신중 → 응급(천식)

신경 · 정신

  • 뇌졸중: 항혈전·자세·의사소통, 최근 6개월 선택적 연기
  • 간질: 응급(경련), phenytoin 치은비대
  • 정신과약: 구강건조·기립저혈압, 삼환계·MAOI ↔ epi 주의

진료 전 체크리스트 · 협진

진료 전 확인

  • 병력 · 복용약 · 알레르기(매 내원 갱신)
  • 활력징후, 질환 조절 상태
  • 최근 입원·시술·사건(MI·뇌졸중·수술)
  • 출혈·감염·스트레스·약물 4축 평가

협진을 고려할 때

  • 미조절·불안정·최근 사건 환자
  • 고위험 침습 시술(다발 발치·외과)
  • 약물 중단/조정·인자보충·스테로이드 커버 판단
  • 불확실하면 주치의에게 확인
원칙 — “이 환자에게 출혈·감염·스트레스·약물 중 무엇이 문제인가?”를 먼저 정리하면 대부분의 결정이 단순해집니다.
