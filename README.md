# 🦷 치과 상용 약물 · 술식별 처방 가이드

치과에서 자주 쓰는 약물을 **체계적으로 분류·정리**하고, **술식·상황별 처방 프로토콜**과
**의사결정 맵(시각화)**으로 한 페이지에 담은 **임상 전문가용 인터랙티브 레퍼런스**입니다.

> 기준: 국내 처방 관행 + 국제 가이드라인(ADA · AHA · AAPD) · 언어: 한국어 · 정리 기준일 2026-06-22

## 열람 방법
- **가장 간단**: `index.html` 파일을 더블클릭하여 브라우저에서 엽니다.
- **로컬 서버**(권장):
  ```bash
  python3 -m http.server 8000
  # 브라우저에서 http://localhost:8000/index.html
  ```
- 다이어그램은 [Mermaid](https://mermaid.js.org/) CDN으로 렌더링되므로 **최초 렌더 시 인터넷 연결**이 필요합니다.
  (표·카드·검색 등 텍스트 콘텐츠는 오프라인에서도 동작)

## 구성
| 섹션 | 내용 |
|---|---|
| ① 술식·상황별 처방 프로토콜 | 발치·매복치·치근단농양·봉와직염·임플란트·**상악동거상술**·치주염·NUG·**TMD**·구내염·칸디다·HSV·건성치조와·예방요법 + **신경병성 통증(하치조신경 손상·BMS)** + **특수 환자군(임산부·항응고제·MRONJ)** 등 21개 시나리오. 각 카드에 **1차 처방(성인) / 약물 알러지 대체 / 소아 용량(체중기반) / 비고**. 부록: **국내 표준 처방 4종 세트** · **신경손상 스테로이드 Tapering(6일) 표** |
| ② 약물 카탈로그 | 14개 대분류(국소마취·진통소염·**신경병성통증**·항생제·항진균·항바이러스·진정·지혈·스테로이드·불소·소독가글·위장보호·근이완·응급) 성분/제품명/적응증/용법/금기/상호작용 표 |
| ③ 알러지 대체 매트릭스 | 페니실린 알러지(경증/중증) × **예방 vs 치료** 구분 대체처방 |
| ④ 소아 용량 | mg/kg 기준 + 1일 최대 + 소아 금기(aspirin·doxycycline·codeine 등) |
| ⑤ 맵·시각화 | 분류 마인드맵(**색상 구분**), 통증 의사결정(ADA 2024), 항생제+알러지 분기, **술식→처방 맵(박스 클릭 시 처방 카드로 이동)**, 전신질환 안전 맵 |
| ⑥ 응급 | 응급상황→약물 맵 + ADA 필수 응급약물 7종 |
| ⑦ 구강내과 복약지도 | 구강 칸디다·점막질환(국소 스테로이드/tacrolimus)·국소마취/진통·구강건조(인공타액·pilocarpine)·소독 함수 — 목적별 제형·사용법. 단종/적응증축소 약물(이모튼 등) 제외, 현행 통용 제제 |

### 인터랙티브 기능
- 상단 **검색창**으로 약물·성분·상황을 실시간 필터링(`/` 포커스 · `Esc` 초기화)
- **맵 박스 클릭 → 해당 처방 카드로 이동·강조**, 색상 범례 클릭으로 시나리오 태그 필터/표 이동
- 목차 네비게이션(**현재 섹션 강조**) · **맨 위로** 버튼 · 반응형(모바일 가로 스크롤 내비) · 인쇄용 레이아웃 · 다크모드(상태 저장)

## ⚠️ 면책 고지
본 자료는 **교육·요약 목적**이며 식약처 허가사항·최신 가이드라인·개별 임상판단을 대체하지 않습니다.
처방 전 **원전과 환자 상태(연령·체중·신/간 기능·병력·복용약)**를 반드시 확인하십시오.
특히 **항생제는 예방요법과 치료요법의 권고가 다릅니다**(예: clindamycin은 예방엔 비권장, 감염 치료엔 사용 가능).

## 유지보수
- 모든 데이터는 `index.html` 내부의 JS 배열(`scenarios`, `catalog`, `allergyRows`, `pedRows`, `emergRows`)에 정의되어 있어 항목 추가/수정이 쉽습니다.
- 완전 오프라인이 필요하면 Mermaid를 CDN 대신 로컬 `mermaid.min.js`로 교체할 수 있습니다.
